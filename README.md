# 医院病案管理系统－－体检、病案共用版

《医院病案管理系统－－体检、病案共用版》是一款集体检、病案于一体的管理软件，对于身体每个部位都有详细登记。在功能方面主要包括:

1.基本信息(既往病史、属性、民族、体检医院建议)

2.体格检查（体格检查信息）

3.病案输入（住院病案录入）

4.分类统计（出院人数统计、按属性统计、按性别统计、按体检结果统计）

# 如何运行本系统?

本系统基于宏达数据库信息管理开发平台开发，下载后，双击文件夹中"dbimp.exe"文件即可完整运行对应的信息管理系统。
